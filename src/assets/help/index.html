Bienvenidos a Galileo


Galileo es una aplicación web que permite la gestión de dispositivos de geolocalización (balizas), su organización, recuperación de posiciones, configuración, posicionamiento en tiempo real y obtención de informes con las posiciones capturadas.

El objetivo de la aplicación web es facilitar la operativa de un número no limitado de agentes (que son el usuario final) y que están agrupados por Unidades. Cada Unidad tiene uno o varios Administradores de Unidad con permisos totales solo para aquello relativo a la unidad a la que pertenecen. Por encima de todos están los SuperAdministradores que tiene acceso a todo el sistema.

Esta documentación de ayuda está diseñada para que pueda aprender rápidamente Galileo como nuevo usuario o mejorar sus conocimientos como usuario habitual.
